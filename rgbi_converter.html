header\(r,g,b)…/(r,g,b)…|
Format Reference
header = any text ending before \ — becomes image metadata comment
\ = start of pixel data
(r,g,b) = one pixel — values 0–255
/ = new row
| = end of image

Example 2×2: My image\(255,0,0)(0,255,0)/(0,0,255)(255,255,255)|
